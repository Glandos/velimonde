How does it work

To get started, pick a city at the top, zooming in before that is useless. Then, you can access the details of a given station by clicking on it. Do NOT try to zoom into oblivion, that's useless. If the city you're looking for is not in the list, well, too bad, it won't show on the map either.

Looks like it's broken?

Try flushing your cache (usually Shift-F5, on the main page). If it doesn't help, please file a bug report on Github including OS and browser! Thanks!
