Comment ça marche / How does it work

Il suffit de sélectionner une ville dans la liste en haut à gauche. Commence pas à zoomer, ya que du vide en dessous, si la ville qui t'intéresse n'est pas dans la liste, ben t'as perdu.

Simply choose your city of interest in the drop-down list in the top left-hand corner. Do NOT try to zoom into oblivion, that's useless. If the city you're looking for is not in the list, well, too bad, it won't show on the map either.

Chez toi ça marche pas ? / Looks like it's broken?

Essaie de vider le cache (en général Maj-F5, depuis la page principale). Sinon prends 2 minutes pour détailler ta config (OS, navigateur, etc.) dans un rapport de bug sur Github ou dans les commentaires du journal Linuxfr ! Merci !

Try flushing your cache (usually Shift-F5, on the main page). If it doesn't help, please file a bug report on Github including OS and browser! Thanks!
