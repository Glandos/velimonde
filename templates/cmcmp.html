Comment ça marche / How does it work

Pour commencer, choisis un ville en haut à gauche, ça ne sert à rien de zoomer avant. Tu peux ensuite accéder aux détails d'une station en cliquant dessus. Commence pas à zoomer, ya que du vide en dessous, si la ville qui t'intéresse n'est pas dans la liste, ben t'as perdu.

To get started, pick a city in the top left-hand corner, zooming in before that is useless. Then, you can access the details of a given station by clicking on it. Do NOT try to zoom into oblivion, that's useless. If the city you're looking for is not in the list, well, too bad, it won't show on the map either.

Chez toi ça marche pas ? / Looks like it's broken?

Essaie de vider le cache (en général Maj-F5, depuis la page principale). Sinon prends 2 minutes pour détailler ta config (OS, navigateur, etc.) dans un rapport de bug sur Github ou dans les commentaires du journal Linuxfr ! Merci !

Try flushing your cache (usually Shift-F5, on the main page). If it doesn't help, please file a bug report on Github including OS and browser! Thanks!
